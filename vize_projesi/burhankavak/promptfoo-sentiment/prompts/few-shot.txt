Aşağıdaki müşteri yorumunun duygu durumunu (pozitif, negatif veya nötr), ve yorumda bahsedilen ana özellikleri olumlu/olumsuz şeklinde JSON formatında döndür. 
Sadece JSON döndür. Başka hiçbir açıklama yapma.

Örnek 1:
Müşteri Yorumu: "Ekranı çok iyi ama şarjı hemen bitiyor."
Çıktı:
{
  "duygu": "karmaşık",
  "özellikler": {
    "ekran": "olumlu",
    "şarj": "olumsuz"
  }
}

Örnek 2:
Müşteri Yorumu: "Harika bir ürün, zamanında geldi."
Çıktı:
{
  "duygu": "pozitif",
  "özellikler": {
    "ürün": "olumlu",
    "kargo": "olumlu"
  }
}

Şimdi şu yorumu analiz et:

Müşteri Yorumu:
{{review}}
