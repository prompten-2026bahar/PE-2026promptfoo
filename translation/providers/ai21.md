# AI21 Labs

[AI21 Labs API'si](https://docs.ai21.com/reference/chat-completion), `jamba-1.5-mini` ve `jamba-1.5-large` gibi AI21 modellerine erişim sağlar.

## API Anahtarı

AI21 Labs'ı kullanmak için `AI21_API_KEY` ortam değişkenini ayarlamanız veya sağlayıcı yapılandırmasında `apiKey` belirtmeniz gerekir.

Ortam değişkenini ayarlama örneği:

```sh
export AI21_API_KEY=your_api_key_here
```

## Model Seçimi

Yapılandırmanızda hangi AI21 modelini kullanacağınızı belirtebilirsiniz. Şu anda aşağıdaki modeller mevcuttur:

1. `jamba-1.5-mini`
2. `jamba-1.5-large`

İşte AI21 modellerini karşılaştıran bir yapılandırma örneği:

```yaml
providers:
  - ai21:jamba-1.5-mini
  - ai21:jamba-1.5-large
```

## Seçenekler

AI21 sağlayıcısı, modelin davranışını özelleştirmek için çeşitli seçenekleri destekler. Bunlar şunları içerir:

- `temperature`: Çıktının rastgeleliğini kontrol eder.
- `top_p`: Çekirdek örneklemeyi (nucleus sampling) kontrol ederek çıktının rastgeleliğini etkiler.
- `max_tokens`: Oluşturulan metnin maksimum uzunluğu.
- `response_format`: JSON çıktısı için `{ type: 'json_object' }`, metin çıktısı için `{ type: 'text' }` olarak ayarlayın.
- `apiKeyEnvar`: API anahtarını içeren bir ortam değişkeni.
- `apiBaseUrl`: AI21 API'sinin temel URL'si.

## Yapılandırma Örneği

İşte AI21 sağlayıcısı için bir yapılandırma örneği:

```yaml
providers:
  - ai21:jamba-1.5-mini
    config:
      apiKey: your_api_key_here
      temperature: 0.1
      top_p: 1
      max_tokens: 1024
      response_format: { type: 'json_object' }
```

Bu yapılandırma, 0.1 sıcaklık, 1 değerinde top-p örnekleme, 1024 tokenlik maksimum çıktı uzunluğu, JSON formatında çıktı ve 1 milyon girdi tokeni başına 0.2$ özel maliyet ile `jamba-1.5-mini` modelini kullanır.

## Maliyet

AI21 modellerini kullanmanın maliyeti modele ve girdi/çıktı token sayısına bağlıdır. Mevcut modeller için maliyetler şöyledir:

- `jamba-1.5-mini`: 1 milyon girdi tokeni başına 0.2$, 1 milyon çıktı tokeni başına 0.4$
- `jamba-1.5-large`: 1 milyon girdi tokeni başına 2$, 1 milyon çıktı tokeni başına 8$

Model için özel bir maliyet belirtmek üzere sağlayıcı yapılandırmasında `cost` seçeneğini ayarlayabilirsiniz.

## Desteklenen ortam değişkenleri

Bu AI21 ile ilgili ortam değişkenleri desteklenmektedir:

| Değişken            | Açıklama                                                        |
| ------------------- | --------------------------------------------------------------- |
| `AI21_API_BASE_URL` | AI21 API'si için kullanılacak temel URL (protokol + ana bilgisayar adı + bağlantı noktası). |
| `AI21_API_KEY`      | AI21 API anahtarı.                                              |
